# Demo Script

## Overview

3-act demo walkthrough for presenting the Expense Reimbursement Manager to assessors.

## Structure

To be populated with the full demo script including:

### Act 1: Setup and Configuration
- Show seed data in lookup tables
- Demonstrate role-based access (Employee vs LM vs HoD views)
- Walk through approval threshold configuration

### Act 2: Claim Lifecycle
- Submit a claim as Employee (show validation rules)
- Approve as Line Manager (show threshold routing)
- Demonstrate self-approval prevention
- Show rejection and resubmission flow

### Act 3: Governance and Reporting
- Walk through audit trail
- Show SLA enforcement
- Demonstrate GL code auto-population
- Review reports and dashboards

Each act includes narration notes, expected screen states, and key talking points.
