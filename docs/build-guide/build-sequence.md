# Build Sequence

## Overview

19-step dependency-ordered build plan for reproducing the Expense Reimbursement Manager from scratch in Zoho Creator.

## Build Steps

To be populated with the complete step-by-step build sequence, including:

1. Application creation and timezone configuration
2. Lookup forms (Departments, Clients, GL_Accounts)
3. Config form (Approval_Thresholds)
4. Transaction form (Expense_Claims) with field dependencies
5. Audit form (Approval_History)
6. Seed data import for all lookup/config tables
7. Role and permission configuration
8. Workflow deployment (On Validate, On Success, On Edit, On Load)
9. Approval process setup (LM + HoD)
10. Scheduled task (SLA Enforcement)
11. Report creation with conditional formatting
12. Dashboard and page configuration
13. Testing and demo preparation

Each step specifies prerequisites, Creator UI navigation path, and verification criteria.
